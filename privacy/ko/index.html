개인정보 처리방침

최종 수정일: 2026년 4월 5일

개요

SnapGPX("본 앱")는 Apple 건강 앱에 기록된 운동을 GPX 파일로 내보낼 수 있게 해줍니다. 사용자의 개인정보는 저희에게 매우 중요합니다. 본 방침은 앱이 어떤 데이터에 접근하며, 어떻게 사용되고, 사용자에게 어떤 선택권이 있는지 설명합니다.

접근하는 데이터

본 앱은 다음과 같은 Apple HealthKit 데이터에 대해 읽기 전용 접근을 요청합니다:

  • 운동 기록 — 활동 유형, 시간, 거리, 소모 칼로리.
  • 운동 경로 데이터 — 운동에 연결된 GPS 좌표, 고도, 타임스탬프.

본 앱은 심박수, 걸음 수, 신체 측정 데이터 또는 운동 및 경로 이외의 HealthKit 데이터 유형에는 접근하지 않습니다.

데이터 사용 방식

  • 운동 및 경로 데이터는 오직 앱 내에서 운동을 표시하고, 사용자가 내보내기를 선택할 때 GPX 파일을 생성하는 데에만 사용됩니다.
  • 모든 데이터 처리는 전적으로 사용자의 기기에서 이루어집니다.
  • 본 앱은 건강 또는 위치 데이터를 외부 서버나 제3자에게 전송하지 않습니다.

Strava 연동

SnapGPX는 운동을 Strava에 직접 업로드할 수 있는 선택적 연동 기능을 제공합니다. 데이터 처리 방식은 다음과 같습니다:

  • SnapGPX는 인증을 위해 Strava의 OAuth 2.0을 사용합니다. 사용자의 Strava 자격 증명은 SnapGPX에서 열람하거나 저장하지 않습니다.
  • OAuth 토큰은 기기의 Keychain에 안전하게 저장됩니다. 토큰은 외부 서버로 전송되지 않습니다.
  • SnapGPX는 Strava에 GPX 데이터를 업로드하기만 합니다. 사용자의 Strava 계정에서 데이터를 읽거나, 다운로드하거나, 저장하지 않습니다.
  • 앱 설정에서 언제든지 Strava 계정 연결을 해제할 수 있습니다. 연결 해제 시 기기에 저장된 모든 토큰이 즉시 삭제됩니다.
  • SnapGPX는 백엔드 서버가 없습니다. 모든 처리는 사용자의 iPhone에서 이루어집니다.

데이터 저장 및 보관

  • 본 앱은 사용자의 건강 데이터를 자체 데이터베이스에 저장하지 않습니다. 앱을 열 때마다 Apple HealthKit에서 직접 데이터를 읽어옵니다.
  • 내보낸 GPX 파일은 사용자가 공유하거나 저장하기로 선택한 위치(예: 파일, AirDrop, 이메일)에만 저장됩니다. 앱은 사본을 보관하지 않습니다.
  • 화면 표시 설정 외에는 세션 간에 앱 내에 데이터가 유지되지 않습니다.

분석

SnapGPX는 익명의 개인정보 친화적 사용 분석을 위해 TelemetryDeck을 사용합니다. 이를 통해 어떤 기능이 사용되는지 파악하고 앱을 개선하는 데 도움이 됩니다.

  • TelemetryDeck은 익명 사용 신호(예: 방문한 화면, 사용한 내보내기 옵션)를 수집합니다. 개인 데이터는 수집되지 않습니다.
  • 고유 사용자 수를 집계하기 위해 해시된 기기 식별자가 사용됩니다. 이 식별자는 사용자 또는 기기와 연결할 수 없습니다.
  • 건강 데이터, 운동 데이터, GPS 좌표 또는 위치 정보는 TelemetryDeck으로 전송되지 않습니다.
  • TelemetryDeck은 앱이나 웹사이트 간에 사용자를 추적하지 않습니다. 광고 SDK는 포함되어 있지 않습니다.
  • 자세한 내용은 TelemetryDeck 개인정보 처리방침을 참조하세요.

제3자 서비스

본 앱은 다음과 같은 제3자 서비스를 사용합니다:

  • TelemetryDeck — 익명 사용 분석(위 참조).
  • Strava — OAuth를 통한 선택적 운동 업로드(위 참조).

광고 또는 추적 SDK는 포함되어 있지 않습니다. 데이터는 제3자에게 판매되지 않습니다.

GPX 내보내기의 위치 데이터

GPX 파일에는 운동 경로의 GPS 좌표가 포함되어 있습니다. 내보낸 후 해당 파일은 사용자의 관리 하에 있습니다. GPX 파일을 공유할 때 운동한 장소가 노출될 수 있으므로 주의하시기 바랍니다.

아동 개인정보 보호

본 앱은 13세 미만의 아동을 대상으로 하지 않습니다. 아동의 개인 정보를 의도적으로 수집하지 않습니다.

사용자 선택권

  • 기기의 설정 > 개인정보 보호 및 보안 > 건강에서 언제든지 HealthKit 접근 권한을 철회할 수 있습니다.
  • 앱 설정에서 언제든지 Strava 연결을 해제할 수 있습니다.
  • 접근 권한을 철회하면 앱에서 운동을 표시하거나 내보낼 수 없게 됩니다.
