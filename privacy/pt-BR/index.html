Política de Privacidade

Última atualização: 5 de abril de 2026

Visão Geral

O SnapGPX (“o App”) permite exportar treinos gravados no Apple Health como arquivos GPX. Sua privacidade é importante para nós. Esta política explica quais dados o App acessa, como eles são utilizados e quais são as suas opções.

Dados que Acessamos

O App solicita acesso somente leitura aos seguintes dados do Apple HealthKit:

  • Registros de treino — tipo de atividade, duração, distância e calorias queimadas.
  • Dados de rota do treino — coordenadas GPS, altitude e timestamps associados aos seus treinos.

O App não acessa frequência cardíaca, contagem de passos, medidas corporais ou quaisquer outros tipos de dados do HealthKit além de treinos e suas rotas.

Como Seus Dados São Utilizados

  • Os dados de treino e rota são utilizados exclusivamente para exibir seus treinos dentro do App e gerar arquivos GPX quando você escolhe exportar.
  • Todo o processamento de dados acontece inteiramente no seu dispositivo.
  • O App não transmite nenhum dado de saúde ou localização para servidores externos ou terceiros.

Integração com o Strava

O SnapGPX oferece integração opcional com o Strava para enviar treinos diretamente. Veja como seus dados são tratados:

  • O SnapGPX utiliza o OAuth 2.0 do Strava para autenticação. Suas credenciais do Strava nunca são vistas ou armazenadas pelo SnapGPX.
  • Os tokens OAuth são armazenados de forma segura no Keychain do seu dispositivo. Nenhum token é enviado a qualquer servidor externo.
  • O SnapGPX apenas envia dados GPX para o Strava. Ele não lê, baixa ou armazena nenhum dado da sua conta do Strava.
  • Você pode desconectar sua conta do Strava a qualquer momento dentro do app. Isso exclui imediatamente todos os tokens armazenados do seu dispositivo.
  • O SnapGPX não possui servidor backend. Todo o processamento acontece no seu iPhone.

Armazenamento e Retenção de Dados

  • O App não mantém um banco de dados próprio dos seus dados de saúde. Ele lê diretamente do Apple HealthKit cada vez que você abre o App.
  • Os arquivos GPX que você exporta são salvos apenas onde você escolher compartilhar ou salvar (ex.: Arquivos, AirDrop, e-mail). O App não mantém cópias.
  • Nenhum dado persiste dentro do App entre sessões, além das suas preferências de exibição.

Análise de Uso

O SnapGPX utiliza o TelemetryDeck para análise de uso anônima e respeitosa da privacidade. Isso nos ajuda a entender quais recursos são utilizados e a melhorar o app.

  • O TelemetryDeck coleta sinais de uso anônimos (ex.: quais telas são visitadas, quais opções de exportação são usadas). Nenhum dado pessoal é coletado.
  • Um identificador de dispositivo em hash é usado para contar usuários únicos. Esse identificador não pode ser vinculado a você ou ao seu dispositivo.
  • Nenhum dado de saúde, dado de treino, coordenada GPS ou informação de localização é enviado ao TelemetryDeck.
  • O TelemetryDeck não rastreia você entre apps ou sites. Nenhum SDK de publicidade está incluído.
  • Saiba mais em política de privacidade do TelemetryDeck.

Serviços de Terceiros

O App utiliza os seguintes serviços de terceiros:

  • TelemetryDeck — análise de uso anônima (veja acima).
  • Strava — envio opcional de treinos via OAuth (veja acima).

Nenhum SDK de publicidade ou rastreamento está incluído. Nenhum dado é vendido a terceiros.

Dados de Localização nas Exportações GPX

Os arquivos GPX contêm coordenadas GPS das rotas dos seus treinos. Depois de exportados, esses arquivos ficam sob seu controle. Tenha cuidado ao compartilhar arquivos GPX, pois eles revelam os locais onde você se exercitou.

Privacidade de Crianças

O App não é direcionado a crianças menores de 13 anos. Não coletamos intencionalmente informações pessoais de crianças.

Suas Opções

  • Você pode revogar o acesso ao HealthKit a qualquer momento em Ajustes > Privacidade e Segurança > Saúde no seu dispositivo.
  • Você pode desconectar o Strava a qualquer momento nas configurações do app.
  • Revogar o acesso significa que o App não poderá mais exibir ou exportar seus treinos.

Alterações nesta Política

Se atualizarmos esta política, a versão revisada será publicada nesta URL com a data atualizada. O uso contínuo do App após as alterações constitui aceitação da política revisada.
